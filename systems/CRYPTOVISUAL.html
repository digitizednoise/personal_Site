Real-Time Global Cryptocurrency Chart
Chart displays the top ten most dominating cryptocurrencies in the market. Visual displays coins ranked in order starting from coin with the highest global percentage.
Prices and current ranking are updated every minute. Current market percentage is updated every 5 minutes. OHLC (red / green price color) is updated every 30 minutes. Data is fetched using NOMICS.COM API.
